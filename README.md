# DNS Wildcard Domain Cleaner

🧹 一个用于 **检测并过滤 DNS 泛解析域名（A / CNAME）** 的异步工具  
适用于安全测试、资产梳理、子域去噪等场景。

---

## ✨ 特性

- 支持 **A / CNAME 泛解析检测**
- 基于 `asyncio + dnspython`，高并发高性能
- 可自定义并发数与测试次数
- 自动过滤无意义的泛解析域名
- 支持单域名 / 批量文件输入
- 进度条实时显示扫描状态
- 支持保留 1 个 `webvpn` 域名作为样本

---

## ⚙️ 安装

```bash
git clone https://github.com/yourname/dns-wildcard-cleaner.git
cd dns-wildcard-cleaner
pip install -r requirements.txt
```
